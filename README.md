# Тьюторська практика: ігрове програмування на Python

Репозиторій містить роботи учнів, які брали участь у майстер класі з ігрового програмування на Python

## Мета
Підвищення інтересу до програмування та розвиток практичних навичок роботи з Python шляхом створення власних інтерактивних програм і простих ігор.

## Опис
У межах практики було розроблено та реалізовано навчальний мінікурс, що поєднує:
- вебінар з основ Python,
- практичний майстер-клас зі створення гри-діалогу,
- фінальний тренінг із самостійною роботою та рефлексією.

Курс побудований на принципах **Inquiry-Based Learning** (навчання через дослідження) та орієнтований на активну участь учнів у процесі навчання.

## Результати
За підсумками практики зафіксовано зростання зацікавленості учнів у програмуванні та позитивну оцінку формату коротких інтерактивних занять.
